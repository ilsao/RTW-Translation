# 译者前言

《Ray Tracing in One Weekend》系列教程以其通俗易懂着称，连夜读完后就萌生了翻译的念头。希望此译本可以帮助想要学习光线追踪技术，但却被语言阻隔的同好。

于是我联系《Ray Tracing in One Weekend》的作者们，与他们表达了创作翻译的意愿。幸运地，我得到他们的鼓励与许可，于是才有这篇译文的出现。

在此，由衷感谢 [Peter Shirley](https://github.com/petershirley), [Trevor David Black](https://github.com/trevordblack), [Steve Hollasch](https://github.com/hollasch) 创作出如此优秀的教程。

如果读者有能力阅读原文，请前往：[https://raytracing.github.io/books/RayTracingInOneWeekend.html](https://raytracing.github.io/books/RayTracingInOneWeekend.html) 阅读原文。译者能力有限，文中可能有多处笔误，恳请各位读者指正。

在译文中，会穿插一些笔者的注释，希望能帮助各位学习这个优秀的教程。

阅读翻译时，若有任何问题欢迎使用邮箱与笔者交流：asciibase64@gmail.com。

如果发现任何翻译不准确，或者观念解释错误的地方，欢迎在 GitHub 上发 Issue：[https://github.com/ilsao/RTW-Translation/issues](https://github.com/ilsao/RTW-Translation/issues)。
